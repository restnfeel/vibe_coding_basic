# 바이브 코딩 베이직에 오신 것을 환영합니다 👋

> 코딩을 몰라도 괜찮습니다. AI와 함께라면 시작할 수 있습니다.

이 가이드는 **AI 코딩 도구를 처음 접하는 분들**을 위해 만들어졌습니다.
어려운 개념보다는 **직접 따라하면서 배우는 것**을 목표로 합니다.

---

## 이 가이드에서 배우는 것

1. 바이브 코딩이 무엇인지 이해하기
2. Cursor 설치하고 첫 AI 대화 해보기
3. AI와 함께 To-Do 앱 만들어보기
4. AI에게 잘 설명하는 방법 익히기
5. 꾸준히 개발하는 나만의 루틴 만들기

첫 프로젝트 완성하기 🚀

AI와 함께 할 일 목록(To-Do) 앱을 처음부터 끝까지 만들어봅니다. 코딩 경험이 없어도 따라올 수 있습니다.

오늘 만들 것​

이런 앱을 만들어봅니다:

  • 할 일을 추가할 수 있습니다
  • 완료된 항목에 체크할 수 있습니다
  • 항목을 삭제할 수 있습니다
  • 브라우저에서 바로 실행됩니다 (별도 설치 불필요)

준비하기​

작업 폴더 만들기​

  1. 원하는 위치에 새 폴더를 만듭니다 예: C:\Users\이름\Desktop\my-todo-app (Windows) 또는 ~/Desktop/my-todo-app (Mac)

  2. Cursor를 열고 File → Open Folder 로 방금 만든 폴더를 엽니다

  3. 왼쪽 파일 탐색기에 빈 폴더가 보이면 준비 완료입니다

1단계: AI에게 앱 만들어달라고 하기​

Ctrl + I (또는 Cmd + I) 로 Composer를 열고 이렇게 입력합니다:

HTML, CSS, JavaScript만 사용해서 할 일 목록 앱을 만들어줘.
파일 하나(index.html)로 완성되어야 해.

기능:
- 입력창에 할 일을 입력하고 "추가" 버튼 클릭하면 목록에 추가
- 각 항목 왼쪽에 체크박스가 있어서 완료 표시 가능
- 완료된 항목은 텍스트에 취소선이 그어짐
- 각 항목 오른쪽에 "삭제" 버튼이 있어서 항목 삭제 가능

디자인:
- 전체 배경은 연한 회색
- 앱 컨테이너는 흰 배경, 둥근 모서리, 그림자
- 버튼은 파란색
- 깔끔하고 심플한 스타일

잠시 기다리면 AI가 index.html 파일을 생성해줍니다.

2단계: 앱 실행해보기​

생성된 index.html 파일을 실행해봅시다.

방법 1: 파일 직접 열기​

  1. 파일 탐색기(Windows) 또는 Finder(Mac)에서 index.html 파일을 찾습니다
  2. 파일을 더블클릭하면 브라우저에서 열립니다

방법 2: Cursor 터미널 사용​

Ctrl + `` 로 터미널을 열고:

start index.html

(Mac이라면 open index.html)

결과 확인​

브라우저에서 앱이 열리면:

  • 입력창에 "오늘 점심 먹기" 같은 할 일을 입력합니다
  • "추가" 버튼을 클릭합니다
  • 목록에 항목이 추가됩니다
  • 체크박스를 클릭해서 완료 표시를 해봅니다
  • 삭제 버튼으로 항목을 지워봅니다

3단계: 원하는 대로 수정하기​

마음에 들지 않는 부분을 바꿔봅시다. AI에게 이렇게 말하면 됩니다.

디자인 바꾸기​

버튼 색상을 초록색으로 바꿔줘.
앱 제목을 "나의 할 일 목록"으로 바꿔줘.

기능 추가하기​

할 일 항목을 드래그로 순서 바꿀 수 있게 해줘.
할 일 개수를 상단에 표시해줘.
예: "총 5개, 완료 2개"
엔터 키를 누르면 할 일이 추가되도록 해줘.

수정 후 저장하기​

AI가 코드를 수정하면 Apply 버튼을 클릭하거나 Ctrl+S 로 저장하고, 브라우저를 새로 고침(F5)하면 변경된 내용이 반영됩니다.

4단계: 오류가 났을 때​

가끔 AI가 만든 코드가 제대로 작동하지 않을 수 있습니다. 이럴 때는:

브라우저 콘솔 확인하기​

  1. 브라우저에서 F12 를 눌러 개발자 도구를 엽니다
  2. Console 탭을 클릭합니다
  3. 빨간 글씨로 오류 메시지가 보이면 복사합니다

AI에게 오류 전달하기​

Cursor의 AI 채팅에 이렇게 붙여넣습니다:

다음 오류가 발생했어. 고쳐줘:

[오류 메시지를 여기에 붙여넣기]

대부분의 경우 AI가 바로 수정해줍니다.

그래도 안 되면​

처음부터 다시 요청해봅니다:

앱이 제대로 작동하지 않아. 처음부터 다시 만들어줘.
이전과 같은 기능인데, 이번에는 더 단순하게 작성해줘.

5단계: 완성된 앱 점검하기​

아래 기능이 모두 동작하는지 확인해봅시다:

  • 할 일 입력 후 "추가" 버튼으로 목록에 추가됨
  • 엔터 키로도 추가됨
  • 체크박스 클릭 시 완료 표시됨 (취소선 등)
  • 삭제 버튼으로 항목 삭제됨
  • 페이지를 새로 고침해도 어느 정도 작동함

모두 확인됐다면, 첫 번째 앱 완성입니다!

도전해볼 것들​

더 해보고 싶다면:

난이도 ⭐ (쉬움)​

앱 배경색을 내가 좋아하는 색으로 바꿔줘. (#색상코드 형식으로)

난이도 ⭐⭐ (보통)​

할 일 항목에 "중요" 표시 기능을 추가해줘.
중요 표시가 된 항목은 빨간색 테두리가 생기도록 해줘.

난이도 ⭐⭐⭐ (도전)​

localStorage를 사용해서 브라우저를 닫아도 할 일 목록이 저장되도록 해줘.

완성! 🎉​

첫 번째 앱을 만들었습니다. 이 과정에서 배운 것:

  • AI에게 원하는 것을 설명하는 방법
  • 코드를 실행하고 테스트하는 방법
  • 오류가 났을 때 AI에게 도움 요청하는 방법
  • 피드백을 통해 앱을 개선하는 방법

이것이 바이브 코딩의 핵심 흐름입니다: 요청 → 결과 확인 → 수정 요청 → 반복

다음 단계​

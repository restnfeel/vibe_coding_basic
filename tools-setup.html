Cursor 설치 & 첫 실행 🛠️

Cursor는 AI가 내장된 코드 에디터입니다. VS Code와 비슷하게 생겼지만, AI가 항상 옆에 있습니다.

Cursor란?​

Cursor는 AI 코딩 도구 중 가장 많이 사용되는 에디터입니다.

  • Microsoft VS Code를 기반으로 만들어졌습니다
  • AI 채팅, 자동 완성, 코드 생성이 기본으로 내장되어 있습니다
  • 무료 플랜으로도 충분히 사용할 수 있습니다
  • Windows, Mac, Linux 모두 지원합니다

1단계: Cursor 다운로드​

  1. 브라우저에서 cursor.com 에 접속합니다
  2. 메인 화면의 "Download" 버튼을 클릭합니다
  3. 운영체제에 맞는 파일이 자동으로 선택됩니다:
    • Windows: .exe 파일
    • Mac: .dmg 파일

2단계: 설치하기​

Windows​

  1. 다운로드된 .exe 파일을 더블클릭합니다
  2. "이 앱이 디바이스를 변경하도록 허용하시겠어요?" → 예
  3. 설치 마법사 안내에 따라 다음 클릭
  4. 설치 완료 후 Cursor가 자동으로 실행됩니다

Mac​

  1. 다운로드된 .dmg 파일을 더블클릭합니다
  2. Cursor 아이콘을 Applications 폴더로 드래그합니다
  3. Applications에서 Cursor를 실행합니다
  4. "인터넷에서 다운로드된 앱입니다" 경고 → 열기 클릭

3단계: 첫 실행 & 계정 설정​

Cursor를 처음 실행하면 설정 화면이 나타납니다.

계정 만들기​

  1. "Sign Up" 클릭
  2. 이메일 주소 입력 후 가입
  3. 이메일로 온 인증 링크 클릭
  4. 다시 Cursor로 돌아오면 로그인 완료
무료 플랜으로 시작하세요

처음에는 무료 플랜(Free)으로 충분합니다. 월 2,000번의 AI 완성 요청이 제공됩니다.

초기 설정 화면​

처음 실행 시 몇 가지 설정을 물어봅니다:

  • 키보드 단축키: VS Code 스타일 선택 (기존 VS Code 사용자) 또는 기본값 유지
  • AI 모델: 기본값 유지
  • 테마: 어두운 테마(Dark) 또는 밝은 테마(Light) 선택

모두 나중에 바꿀 수 있으니 일단 기본값으로 진행해도 됩니다.

4단계: 화면 구성 이해하기​

Cursor를 열면 이런 화면이 보입니다:

┌─────────────────────────────────────────────┐
│  파일 탐색기  │      코드 편집 영역          │
│  (왼쪽 패널)  │                              │
│               │                              │
│  - 폴더       │   // 여기에 코드가 표시됩니다│
│  - 파일       │                              │
│               │                              │
├───────────────┴──────────────────────────────┤
│  터미널 (하단, Ctrl+` 로 열기)               │
└──────────────────────────────────────────────┘

주요 영역:

  • 왼쪽 파일 탐색기: 프로젝트 파일들을 볼 수 있습니다
  • 가운데 편집 영역: 코드를 보고 편집하는 곳입니다
  • AI 채팅 패널: 오른쪽 또는 단축키로 열 수 있습니다

5단계: AI에게 첫 말 걸어보기​

이제 AI와 대화해봅시다!

AI 채팅 열기​

  • 단축키: Ctrl + L (Windows) / Cmd + L (Mac)
  • 또는 오른쪽 상단의 채팅 아이콘 클릭

채팅 창이 열리면 입력창이 나타납니다.

첫 번째 대화 해보기​

입력창에 이렇게 입력해봅시다:

안녕! 나는 오늘 처음 Cursor를 사용해보는 초보자야.
간단한 "Hello, World!"를 출력하는 HTML 파일을 만들어줘.

잠시 기다리면 AI가 코드를 생성해줍니다. 예를 들면:

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>첫 번째 페이지</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>바이브 코딩에 오신 것을 환영합니다!</p>
</body>
</html>

코드 적용하기​

AI가 코드를 생성하면:

  1. 코드 블록 오른쪽 위의 "Apply" 또는 "Copy" 버튼 클릭
  2. 새 파일에 붙여넣기 (Ctrl+N 으로 새 파일 생성)
  3. index.html 이름으로 저장 (Ctrl+S)
  4. 파일을 브라우저로 열면 결과를 볼 수 있습니다

6단계: AI Composer 사용해보기​

Composer는 Cursor의 가장 강력한 기능입니다. 파일을 직접 생성하고 편집해줍니다.

Composer 열기​

  • 단축키: Ctrl + I (Windows) / Cmd + I (Mac)

Composer로 파일 만들어보기​

Composer에 이렇게 입력해봅시다:

현재 폴더에 index.html 파일을 만들어줘.
내용은 "안녕하세요! AI 코딩을 시작합니다."라는 제목과
간단한 소개 텍스트가 있는 웹 페이지야.
배경색은 연한 파란색으로 해줘.

Composer가 파일을 직접 생성해줍니다!

자주 쓰는 단축키 모음​

기능WindowsMac
AI 채팅 열기Ctrl + LCmd + L
Composer 열기Ctrl + ICmd + I
터미널 열기/닫기Ctrl + `` Cmd + ``
파일 저장Ctrl + SCmd + S
새 파일Ctrl + NCmd + N
파일 열기Ctrl + OCmd + O

문제 해결​

Cursor가 실행되지 않아요​

  • Windows: 관리자 권한으로 실행해보세요
  • Mac: 보안 설정에서 Cursor 허용 → 시스템 환경설정 → 보안 및 개인 정보 보호

AI 응답이 느려요​

  • 인터넷 연결 상태를 확인하세요
  • 무료 플랜 사용량 한도를 확인하세요 (월 2,000회 제한)

한국어로 대화가 안 돼요​

  • 한국어로 질문하면 한국어로 답변해줍니다. 자연스럽게 한국어로 대화해도 됩니다!

체크리스트​

설치가 완료됐는지 확인해봅시다:

  • Cursor 설치 완료
  • 계정 생성 및 로그인
  • AI 채팅 창 열기 성공 (Ctrl+L / Cmd+L)
  • AI에게 첫 인사 해보기
  • HTML 파일 생성해보기

모두 체크됐다면 준비 완료입니다!

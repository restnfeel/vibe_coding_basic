다음으로 배울 것들​

더 복잡한 앱 만들기​

  • 백엔드 서버 연동 (데이터를 서버에 저장)
  • 여러 페이지로 구성된 웹사이트
  • 사용자 로그인/회원가입

개발 도구 더 잘 활용하기​

  • 반복 작업 자동화하기
  • 비용 효율적으로 AI 쓰는 법

협업하기​

  • GitHub으로 코드 공유하기
  • 다른 개발자와 함께 작업하기
  • 코드 리뷰 요청하기

효율화 가이드로 계속하기​

더 빠르고 효율적으로 개발하고 싶다면, 다음 가이드가 준비되어 있습니다.

⚡ 바이브 코딩 효율화 가이드​

중급자를 위한 이 가이드에서는:

  • Quality · Speed · Cost 균형 최적화 전략
  • Cursor + Opencode + vLLM 도구 조합
  • 비용을 80% 줄이는 실전 워크플로우
  • 도메인 → 태스크 → 스킬 아키텍처 설계
  • 여러 AI 모델을 상황에 맞게 선택하는 법

을 배울 수 있습니다.

베이직 이후 추천 실습 프로젝트​

레벨 1 (지금 당장 가능)​

  • 가계부 앱: 수입/지출 기록, 카테고리별 분류, 월별 합계
  • 독서 기록 앱: 책 제목, 저자, 별점, 한줄평 저장
  • 타이머 앱: 뽀모도로 타이머, 알림 기능

레벨 2 (2~4주 후)​

  • 날씨 앱: 실제 API 연동, 위치별 날씨 표시
  • 퀴즈 앱: 문제 출제, 점수 계산, 결과 화면
  • 링크 저장소: 나중에 읽을 링크 수집, 태그 분류

레벨 3 (1~2개월 후)​

  • 블로그: 글 작성, 카테고리, 댓글
  • 포트폴리오 사이트: 나의 작업물 전시
  • 간단한 쇼핑몰: 상품 목록, 장바구니

계속 성장하는 법​

매일 조금씩​

이미 5챕터에서 배웠지만, 다시 한번 강조합니다. 매일 30분의 꾸준함이 가장 강력한 성장 방법입니다.

만든 것을 공유하기​

GitHub Pages로 배포해서 다른 사람들이 사용할 수 있게 해보세요. 실제 사용자가 생기면 더 열심히 만들게 됩니다.

커뮤니티 참여하기​

혼자 하는 것보다 같이 하면 더 빠릅니다. AI 코딩 관련 커뮤니티에서 다른 분들의 경험을 나눠보세요.

마무리 메시지​

이 가이드를 끝까지 읽고 따라오신 여러분에게 한 마디:

시작이 반입니다. 그리고 여러분은 이미 시작했습니다.

처음 며칠은 AI가 만들어준 코드가 마법처럼 느껴질 것입니다. 한 달이 지나면 AI에게 더 잘 설명하게 됩니다. 세 달이 지나면 스스로 아이디어를 가지고 앱을 완성하게 됩니다.

지금 당장 완벽하지 않아도 됩니다. 계속 만들다 보면 됩니다.
